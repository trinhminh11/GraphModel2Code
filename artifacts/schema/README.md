# Schema Design

**Owner: Human (Senior AI/Backend Engineer)**

This folder holds JSON schema drafts and specs for the graph data model.

## Status

- [ ] Node definition schema
- [ ] Edge definition schema
- [ ] Graph container schema
- [ ] Layer registry format
- [ ] Visual style properties

## How to Use

Drop your schema drafts here as `.json` or `.md` files. The AI agent will read them and adapt the frontend types and components to match.

Until the final schema is provided, the frontend uses a self-contained mock schema (in `frontend/src/types/`) that covers all UI-facing fields.
